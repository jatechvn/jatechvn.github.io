<a id="introduction"></a>
## 소개

본 프로젝트는 전문적인 프로젝트 구조에 따라 표준화되어 있으며, Git을 통한 효율적인 소스 코드 관리를 지원합니다.

<a id="features"></a>
## 주요 기능

- 빠른 설정 및 전문가 수준의 패키징.
- 유지 관리와 확장이 용이한 가벼운 구성.
- 포함된 스크립트를 통한 자동 Git 푸시 워크플로우.

## 디렉토리 구조

```text
jatechvn.github.io/
├── main.py                    # 애플리케이션 진입점
├── run.bat                    # Windows용 빠른 시작 스크립트
├── requirements.txt           # 의존성 목록
├── config.ini                 # 간단한 설정 파일
├── config.json                # 구조화된 JSON 설정 파일
├── .gitignore                 # Git 제외 패턴
├── README.md                  # 프로젝트 문서 (본 파일)
├── LICENSE                    # 라이센스 파일
├── ABOUT.txt                  # 프로젝트에 대한 짧은 설명
├── git_push.bat               # GitHub 자동 푸시 스크립트
│
├── logs/                      # 애플리케이션 로그
│
├── modules/                   # 비즈니스 로직을 포함하는 핵심 모듈
│   # 비즈니스 로직을 포함하는 핵심 모듈
│
└── assets/                    # 애플리케이션 정적 리소스
```
