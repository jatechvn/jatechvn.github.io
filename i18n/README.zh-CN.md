<a id="introduction"></a>
## 项目简介

项目按照专业的项目结构进行标准化配置，支持通过 Git 进行高效的源代码管理。

<a id="features"></a>
## 核心特性

- 快速部署与专业化打包。
- 轻量化配置，易于维护与扩展。
- 附带的脚本支持自动推送代码至 Git 仓库。

---

## 目录结构

```text
jatechvn.github.io/
├── main.py                    # 应用程序入口点
├── run.bat                    # Windows 快速启动脚本
├── requirements.txt           # 所需依赖库列表
├── config.ini                 # 简单键值配置文件
├── config.json                # JSON 结构化配置文件
├── .gitignore                 # Git 忽略文件配置
├── README.md                  # 项目说明文档 (本文件)
├── LICENSE                    # 开源许可证文件
├── ABOUT.txt                  # 项目简短描述
├── git_push.bat               # 自动推送代码至 GitHub 的脚本
│
├── logs/                      # 应用程序运行日志
│
├── modules/                   # 核心业务逻辑模块
│   # 核心业务逻辑模块
│
└── assets/                    # 静态资源目录
```
